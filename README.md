# React - React-Coin-list

## 사이트 설명
React와 Typescript를 적용하여 만든 Coin 암호화폐 정보 사이트 입니다

## 적용기술
1.React-query  
2.TypeScript  
3.react-helmet  
4.recoil  
5.axios  
6.styled-components  
7.react-dom  
    - nest-routes 적용 : chart / price
